Week 04 Notes

    Ch 6: DOM

  1. childNodes: property is a list of all the nodes that are children of the node concerned; will include any text nodes, and since whitespace is treated as a text node, there will often be empty text nodes in this collection
  2. children: property only returns anyelementnodes that are children of that node, so will ignore any text nodes
  3. firstChild: property returns the first child of a node; can often be a text node, even if it’s just an empty string generated by some whitespace
  4. lastChild: property returns the last child of a node; can often be a text node, even if it’s just an empty string generated by some whitespace
  5. other properties: nextSibling, previousSibling, parentNode, etc
  6. getAttribute() method returns the value of the attribute provided as an argument
